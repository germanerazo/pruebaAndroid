# pruebaAndroid
This project allows chatting between users through internal messages.

Requerimientos:
Android studio
Codigo fuente

componentes utilizado
Firebase
Kotlin


This has the following characteristics:

User register
Forgot your password
Login
Create contacts
Edit contacts
delete contacts
Chat between contacts
